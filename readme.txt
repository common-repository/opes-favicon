Opes Favicon allows you to add and manage favicons & icons on your WordPress website.

== Description ==

Opes Favicon allows you to add and manage favicons & icons on your WordPress website using WordPress media library and custom URL.

<strong style="font-size: 18px;">Completely new! Check it!</strong>

Major features in Opes Favicon include:

* Favicon URL validation
* Automatic generation all favicons & icons
* Support mobile devices
* All favicons & icons are set in the root

== Installation ==

* Upload the Opes Favicon plugin to your blog and Activate it
* Go to "Appearance->Opes Favicon" and set your website favicons & icons
